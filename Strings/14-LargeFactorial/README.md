## Problem : Large Factorial (Medium)
Given a number *A**. Find the fatorial of the number.

**Input Format:**
First and only argument is the integer A.

**Output Format:**
Return a string, the factorial of A.


**Example 1:**
```
Input:
A = 2

Output:
2

Explanation:
2! = 2 
```

**Example 2:**
```
Input:
A = 3

Output:
6

Explanation:
3! = 6 
```


**Constraints:**
<li>1 <= <b>A</b> <= 100</li>
